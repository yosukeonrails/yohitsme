    Who is me?

    My name is Yosuke Hishinuma and I am a fullstack javascript developer , a part-time singer and a full time joker. But you can just call me Yoh.

    Bio

    Began my first experiment with programming in the summer of 2015, when couple very trusted friends of mine suggested that I learn how to code. Did not know anything at that point but I chose to believe that anything is learnable. Started with HTML, CSS and client side Javascript , later experimented with few different languages like Ruby and Python until landing on Javascript for server side programming. Today I do full stack programming with emphasis on React JS.

    Technologies

    Projects were a great combination of React/Redux JS on the front-end , NodeJS and MongoDB on the back-end.
